# Documentation

This directory contains detailed guidance for using and extending the reusable
workflows and composite actions in this repository.

Start here:
- `docs/workflows.md` for CI/PR gate/deploy workflow inputs, ordering, and examples.
- `docs/presets.md` for stack-specific and E2E presets (Node/React/Playwright/Cypress).
- `docs/actions.md` for composite actions (semver compare, release tag guard).
- `docs/examples.md` for common usage patterns (monorepos, E2E servers, Docker + E2E).
